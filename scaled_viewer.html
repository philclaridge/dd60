DD60 Five-Parameter Display System

1. Character Scale (CDC Native): Hardware-authentic CDC 6602 character sizing - 1× (7×7px), 2× (14×14px), 4× (28×28px). Line width remains constant.

2. Canvas Scale (Display Resolution): Logical canvas size - 1 (512px), 2 (1024px), 4 (2048px), 8 (4096px). Currently fixed at Canvas Scale 1.

3. Device Scale (Future): Device pixel ratio for Retina/HiDPI displays. Not yet implemented.

4. Bit Depth (Color Mode): 1-bit (monochrome/authentic) or 8-bit (RGBA/modern). Currently using mixed modes.

5. Artifact Level (Future): CRT effects like phosphor blur, XY distortion, beam convergence. Placeholder for authentic visual characteristics.
